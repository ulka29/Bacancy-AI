## 🔵 CNN with Pytorch

**Dataset name:** CIFAR-10

**Dataset link:** [https://www.kaggle.com/competitions/cifar-10](https://www.kaggle.com/competitions/cifar-10)

**Notebook GitHub:** [https://github.com/ulka29/Bacancy-AI/blob/main/DL/cnn-with-pytorch.ipynb](https://github.com/ulka29/Bacancy-AI/blob/main/DL/cnn-with-pytorch.ipynb)

**Notebook Kaggle:** [https://www.kaggle.com/code/ulkapatel/cnn-with-pytorch](https://www.kaggle.com/code/ulkapatel/cnn-with-pytorch)

### Training Configuration

* Optimizer: Adam (α = 0.001)
* Loss: Cross-Entropy
* Hardware: CUDA-enabled (GPU) support

### Data Preprocessing

* Resizing: Images resized to 32 × 32 pixels
* Normalization: Mean and Std set to 0.5 for all 3 channels (RGB) to scale pixel values to [-1, 1]
* Splitting: 45,000 images for training and 5,000 for validation
